GOAL: Level up in React to build impactful projects and stand out as a developer.
1
Setup & Core Basics
  • Environment Setup
    • Install Node.js + npm
    • Create project using Vite or Create React App (CRA)
    • Understand folder structure
    • Learn JSX (HTML in JS)
2
Core Building Blocks
  • Components & Props
    • Functional components
    • Pass data with props
    • Reuse and structure components
  • State & Events
    • Manage internal data with useState
    • Handle button clicks, input changes
  • React Router
    • Create multiple pages
    • Navigate with Link and useNavigate
    • Use dynamic routes with useParams
  • useEffect & API Calls
    • Perform side effects with useEffect
    • Fetch data from APIs
    • Handle loading and error states
3
Intermediate Concepts
  • Context API
    • Share state globally across components
    • Avoid prop drilling
  • State Management
    • Lifting State Up
    • Share state between sibling components
  • Advanced Data Fetching
    • Use fetch() or axios
    • Handle loading, success, and error states
    • Build reusable fetch logic
